## Goal
Set up the trafficlab/ WASM project: Rust crate with wasm-pack, index.html canvas shell, Makefile targets (build, serve, release), and web/ output directory structure

## Acceptance Criteria
- [ ] `trafficlab/` Rust crate compiles to WASM via `wasm-pack build` without errors
- [ ] `make build-trafficlab` produces output under `trafficlab/pkg/` and `trafficlab/web/`
- [ ] `make serve-trafficlab` starts a local HTTP server and serves `index.html` on localhost
- [ ] `index.html` loads the WASM module and renders a blank canvas without console errors
- [ ] `make release-trafficlab` produces a minified deployable artifact in `trafficlab/dist/`
