## Goal
Reduce global mutex contention by moving persistence handle ownership out of ServerState and locking it independently.

## Acceptance Criteria
- [ ] Move persistence handle ownership out of `ServerState` into a dedicated shared lock (`Arc<Mutex<Option<Persist>>>`).
- [ ] Route AOF logging and `SAVE`/`BGSAVE` command persistence calls through the dedicated persistence lock.
- [ ] Preserve shutdown snapshot behavior and persistence-not-configured error semantics.
- [ ] Validate with unit tests and MuonTickets board validation.
