## Goal

Draw a **persistent leaderboard panel** in the top-right corner of the IDM Live
canvas showing the top-5 rounds ranked by survival time.  The panel is always
visible during active play so players can track progress across rounds.

## Acceptance Criteria

- [ ] `IDMRenderer` gains a `drawLeaderboard(rows)` method, where `rows` is the
      array returned by `getLeaderboard()` (from T-000123):
      `[{rank, id, duration_s, cars_crossed, max_wait_s}, ...]`
- [ ] Panel is positioned in the **top-right** corner of the canvas with 10px
      margin from the edges.
- [ ] Panel shows a header row: `🏆 Top Runs`
- [ ] Each data row displays: `#rank  Xs  N cars  W s wait`
      (right-aligned numbers, e.g. `#1  47.3 s  12 cars  3.1 s wait`)
- [ ] The **current round's row** (if it appears in the top-5) is highlighted
      with a distinct background colour (e.g. gold/amber tint).
- [ ] Panel background is semi-transparent dark (`rgba(0,0,0,0.55)`), text is
      white, using the same monospace font already used in `_drawStats`.
- [ ] If `rows` is empty or null the panel is not drawn (first round before any
      collision).
- [ ] `drawLeaderboard` is called from the RAF callback in `app.js` after
      `renderer.draw(sim)`, passing the cached leaderboard data (updated once per
      round-end, not every frame).
- [ ] Canvas width stays at 900 px; the panel must fit within that width without
      overlapping the intersection graphic.

## Implementation plan

1. Add `drawLeaderboard(rows, currentRoundId)` to `IDMRenderer` in `idm.js`.
2. Compute panel dimensions based on number of rows (header + up to 5 data rows,
   each 18 px tall, plus 8 px padding top/bottom and sides).
3. Use `ctx.save()` / `ctx.restore()` to isolate transform context.
4. Draw rounded-rect background with `rgba(0,0,0,0.55)`.
5. Render header text in bold, data rows in normal weight.
6. Highlight matching `currentRoundId` row with `rgba(255,200,0,0.25)` fill.
7. In `app.js` `initIDM()`:
   - keep a `leaderboardRows = []` variable
   - after `saveRoundScore`, call `leaderboardRows = getLeaderboard(cache)`
   - pass `leaderboardRows` and `'game:' + roundNumber` to `renderer.drawLeaderboard()`
      each RAF frame.

## Test plan

- Run until 5+ rounds have completed; panel should show exactly 5 rows.
- The most recent round (if top-5) should be highlighted.
- Resize the browser window to verify the panel stays inside the canvas.
- Check that the intersection road and vehicles are not obscured by the panel
  (top-right quadrant of canvas is mostly road surface / legend area).
