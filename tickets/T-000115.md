## Goal
Add selectable traffic modes: Normal, Rush Hour, Festival Traffic, Rain Traffic (each with distinct arrival distributions and signal timing). Add country rule presets: India (free left turns, motorcycle lane splitting) and US (strict lanes, protected turns). Modes are selectable from the UI and immediately reset the simulation with new parameters

## Acceptance Criteria
- [ ] All four traffic modes produce measurably different arrival rates and queue behavior
- [ ] India preset enables free left turns and motorcycle lane-splitting behavior
- [ ] US preset enforces strict lanes and protected-only turns
- [ ] Switching mode or country preset resets both simulations from tick 0
- [ ] Cache hit ratio changes between modes (verifying the cache adapts to different patterns)
- [ ] Mode/preset selector is visible in the UI and labeled clearly

## Test Plan
- Unit test: run Normal vs Rush Hour 1,000 ticks — assert Rush Hour vehicles/min is >= 2x Normal
- Unit test: India preset — assert free-left-turn vehicles bypass signal queue
- Unit test: US preset — assert protected-only turns wait for explicit green turn signal
- Visual test: switch modes mid-demo, verify reset and re-run

## Notes
This feature amplifies demo persuasiveness for global audiences (India traffic especially relevant for launch).
