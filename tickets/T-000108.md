## Goal
Implement the simulation loop: vehicle spawn (car/motorcycle/truck/bus/auto-rickshaw at configurable arrival rates), traffic signal state machine (configurable cycle durations), lane queue model, vehicle movement engine (straight/left/right turns), and per-tick statistics (wait times, throughput, ticks/sec)

## Acceptance Criteria
- [ ] Vehicle spawn produces correct type distribution (car 60%, motorcycle 20%, truck 10%, bus 5%, auto-rickshaw 5%) across 10,000 spawns ±2%
- [ ] Signal state machine cycles through red/green/yellow correctly with configurable durations
- [ ] Lane queues grow under red signals and drain under green; no vehicle teleportation
- [ ] Per-tick stats (ticks/sec, avg wait time, vehicles processed) are computed and accessible
- [ ] Simulation runs headlessly for 10,000 ticks without panic or undefined behavior
- [ ] `cargo test` passes all simulation unit tests

## Test Plan
- Unit test: spawn 10,000 vehicles, assert type distribution within 2% of target
- Unit test: run signal through 5 full cycles, assert state sequence and timing
- Unit test: fill N/S/E/W queues to capacity, step signal to green, assert queues drain
- Headless benchmark: 10,000 ticks, assert no panic, ticks/sec > 1,000 on dev machine
