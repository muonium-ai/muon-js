## Goal
Reduce global lock contention by reading script runtime config once per client and removing per-command ServerState mutex locks in handle_client.

## Acceptance Criteria
- [ ] In `handle_client`, read `script_runtime` config from shared `ServerState` once at connection setup and reuse local state during command loop.
- [ ] Remove per-command `state.lock().await` in both normal and `EXEC` command paths while preserving command semantics.
- [ ] `cargo check --bin mini_redis --features mini-redis` succeeds.
- [ ] `cargo test --lib` remains green.
