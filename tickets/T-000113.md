## Goal
Implement auto-run: simulation starts immediately on page load, no click required. Add demo page polish: product header/tagline, legend, social share link. Add Makefile deploy target to publish to GitHub Pages or static hosting. Verify the demo runs correctly in Chrome, Firefox, and Safari. Record a shareable demo URL

## Acceptance Criteria
- [ ] Simulation begins rendering within 3 seconds of page load — no button click needed
- [ ] Page includes: "MuonCache TrafficLab" header, tagline, vehicle color legend
- [ ] `make deploy-trafficlab` publishes to GitHub Pages (or configured static host) without manual steps
- [ ] Demo URL is accessible publicly and loads within 5 seconds on a standard connection
- [ ] No console errors in Chrome 120+, Firefox 120+, Safari 17+

## Test Plan
- Open demo URL in each of three browsers — verify auto-run within 3s
- Check browser console for errors — must be zero
- Run Lighthouse performance audit — score >= 80
- Share URL to a second machine to verify it loads without auth
